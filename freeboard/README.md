# 중고마켓

## sanghyeon.shop

## 프로젝트 소개

CRUD 게시판 프로젝트입니다. 비회원이 쓸 수 있는 자유게시판과, 회원이 쓸 수 있는 중고마켓을 만들었습니다. 중고마켓은 회원 가입 후 누구나 물건을 올릴 수 있으며, 포인트 충전 후 구매하도록 구현했습니다.

## 프로젝트 스택
